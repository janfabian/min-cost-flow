You just created your first node. Now click on the node and you will see its edit dialog window.

In general all the nodes are classified only by its variable b. It says if the node is

  • sink (b < 0)
  • source (b > 0)
  • ordinary node (b = 0)

Your task

By dragging from one node to another create following edges.

  • From the source node (b = 4) to your newly added node
  • From your newly added node to the sink node (b = -4)
